# Brow Best - лендинг студии бровей и ресниц

Одностраничный лендинг для бьюти-студии **Brow Best** (Новосибирск). Цель страницы -
запись клиента: форма, кликабельный телефон и мессенджеры. Кейс для портфолио
веб-разработчика.

🔗 **Демо:** https://USERNAME.github.io/REPO/
*(замените `USERNAME` и `REPO` на свои после публикации - см. ниже)*

## Стек

- Чистый **HTML / CSS / JS**, без фреймворков и сборщиков. Вся страница - один файл
  `index.html` (разметка + `<style>` + `<script>` IIFE).
- Шрифты - Google Fonts (`Cormorant Garamond` + `Manrope`).
- Иконки - Tabler Icons (вебфонт через CDN).
- Фото работ - локальные, в папке `images/`.
- Анимации появления на `IntersectionObserver`, карта грузится лениво по клику.
  Без тяжёлых библиотек и автоплея.

## Структура

| Файл / папка | Назначение |
|---|---|
| `index.html` | Вся страница: hero, услуги, галерея, мастер, отзывы, запись, контакты. |
| `images/` | Фото работ (`work-01..15.png`) и портрет мастера (`Olesja.jpg`). |
| `.nojekyll` | Отключает обработку Jekyll на GitHub Pages (статика отдаётся как есть). |

## Локальный запуск

Это статический сайт, сборка не нужна. Достаточно поднять любой локальный сервер:

```bash
# Python 3
python -m http.server 4321
# затем открыть http://localhost:4321
```

Или просто открыть `index.html` в браузере.

## Деплой

Сайт опубликован на **GitHub Pages** из ветки `main` (корень репозитория).
Любой пуш в `main` автоматически обновляет страницу.
